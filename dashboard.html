Dashboard Features

This dashboard includes the following interactive elements:

  • State Filter: Select individual states or multiple states to analyze consumption patterns
  • Region Filter: Compare consumption across regions (Northern, Western, Southern, Eastern, North-Eastern)
  • Date Range Filter: Focus on specific time periods (e.g., pre-lockdown, during lockdown, post-lockdown)
  • Top N/Bottom N Filters: Quickly identify highest and lowest consuming states
  • Year Filter: Compare 2019 vs 2020 consumption patterns
